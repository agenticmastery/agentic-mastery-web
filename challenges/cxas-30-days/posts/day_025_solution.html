Solution: Python Script to List Applications

Here is the complete Python script to list applications in your project. This script follows best practices by using an environment variable for the Project ID and including error handling.

import os
from cxas_scrapi import Apps

def list_cxas_apps():
    """Lists all CXAS applications in the project specified by environment variable."""
    # Get project ID from environment variable
    project_id = os.environ.get("GCP_PROJECT_ID")
    
    if not project_id:
        print("Error: GCP_PROJECT_ID environment variable not set.")
        print("Please set it using: export GCP_PROJECT_ID='your-project-id'")
        return

    try:
        print(f"Connecting to CXAS and listing apps for project: {project_id}...")
        apps_client = Apps()
        apps = apps_client.list_apps(project_id=project_id)

        if not apps:
            print("No applications found in this project.")
            return

        print("\nAvailable Applications:")
        for app in apps:
            print(f"- Name: {app.display_name}")
            print(f"  ID:   {app.name}")
            print("-" * 20)

    except Exception as e:
        print(f"An error occurred: {e}")
        print("Ensure you have run 'gcloud auth application-default login'.")

if __name__ == "__main__":
    list_cxas_apps()

Key Steps Explained

  • Importing the Client: We import Apps from cxas_scrapi. This is the core client for application management.
  • Authentication: We do not pass explicit credentials. The client automatically looks for Application Default Credentials (ADC) configured in your environment.
  • Environment Variables: We use os.environ.get("GCP_PROJECT_ID") to avoid hardcoding sensitive or environment-specific data.
